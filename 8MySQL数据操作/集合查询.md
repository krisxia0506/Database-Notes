# UNION
【例1】查询专业号1102的学生和所有女生。
方法一:
UNION查并集，去掉重复行
```sql
SELECT* FROM S WHERE Zno=‘1102'
UNION
SELECT * FROM S WHERE Ssex=‘女";
```
方法二:
不去掉重复行
```sql
SELECT DISTINCT *
FROM S
WHERE Zno=‘1102’OR Ssex=‘女";
```
