# 1.域
是一组具有相同数据类型的值的集合
例如：整数 、实属 、{男，女}

# 2.笛卡尔积
笛卡尔积可以表示为一个二维表
例如：有D1（a，b，c）和D2（1,2）两个域
则D1、D2的笛卡尔积为：
D1XD2={(a,1)，(a,2)，(b,1)，(b, 2)，(c,1)，(c,2)}
![[笛卡尔积二维表.png]]
## 元组
笛卡尔积中的每一个元素（d1,d2,d3....dn）

## 分量
笛卡尔积元素中的每个值di

# 3.关系
表示为R(D1,D2..... Dn),
+ R为关系名;
+ n是关系的目或度(Degree);
+ 当n=1时为单元关系;n=2时为二元关系。
+ 关系中的每个元素是关系中的元组，用t表示.
+ 关系是笛卡尔积的有限子集。
