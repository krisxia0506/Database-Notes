# 1.实体
客观存在并可相互区别的事物

# 2.属性
实体所具有的的某一特性称为*属性*
一个实体有若干个属性来描述
例如：学生（*学号，姓名，性别*）

# 3.码
能***唯一标识实体***的属性称为码
*可由一个或多个其值能唯—标识该关系模式中任意元组的属性组成*

# 4.实体型
用实体名和属性名的集合来抽象的同类实体
例如：*学生（学号，姓名，性别）*

# 5.实体集
同类型实体的集合
例如：*全体学生*

# 6.域
指实体中属性的取值范围
例如：考试成绩的取值通常为0-100

# 7.联系
现实世界中事物内部及事物间的联系，在信息世界中反映为实体内部的联系和实体间的联系
三种类型:1:1、1:n、m:n
